Reminders

  1. Medicaid check for IL, MO, and WA
  2. Reminders

Please be reminded of the following:

  1. If the account is in Rejected Referral status or TPL status, we can skip it we don't need to run RPA. Those two statuses indicate that the patient already has insurance.
  2. If the visit is Hold, No or Pending Approval, run RPA.
    • If the visit is Hold, you will update the status detail to manual review, note and scan.
    • If the visit is No, you will status override to put the visit back into hold status, manual review. Then note and scan (upload).
    • If the visit is Pending Approval, you will not change the status when you find Medicaid. You will enter a note and scan the POS.
  3. If the Name, DOB match PACE and RPA but the Address is different, we can proceed with the process and put note on PACE that the address does not match.
  4. We do NOT consider Family Planning as active Medicaid coverage.

NOTE:
